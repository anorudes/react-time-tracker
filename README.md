React Time Tracker
=========================

## Demo
[http://anorudes.github.io/react-time-tracker](http://anorudes.github.io/react-time-tracker)

## Installation
```
$ npm install
```

## Run app
```
$ npm start
```
And open in browser: [http://localhost:3000](http://localhost:3000)

## Build
```
$ npm run build
```
